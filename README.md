# Arthur Jeu - Ultimate ⚔️

Bienvenue dans **Arthur Jeu - Ultimate**, un jeu de combat en 2D dynamique inspiré de *Super Smash Bros*, développé entièrement en Python avec **Pygame**. 

Ce projet propose une expérience de combat moderne avec des mécaniques fluides, un système de manches compétitif et des effets sonores procéduraux.

## 🌟 Fonctionnalités Principales

- **Mode Versus Compétitif** : Des matchs intenses en 5 manches gagnantes avec un chronomètre de 2 minutes par round.
- **Rotation Automatique des Cartes** : À chaque KO, le stage change automatiquement pour offrir une expérience variée (Caverne, Ciel, Ville Cyber, Vide de Cristal).
- **Personnages Uniques** :
    - **Le Chevalier** : Maître de l'épée avec des capacités de téléportation.
    - **Le Mage** : Utilise la magie, des boules de feu et des transformations météores.
- **Système de Skins** : Personnalisez vos combattants avec différentes variations de couleurs.
- **Audio Rétro Procédural** : Sons de saut, d'attaque et d'impact générés en temps réel (pas de fichiers externes requis !).
- **HUD Moderne** : Affichage clair des dégâts (%), du temps restant et des scores.
- **Multijoueur** : Support pour le jeu local et une base pour le jeu en réseau (IP Locale).

## 🎮 Commandes

### Joueur 1 (P1)
- **Déplacement** : `Q` / `D`
- **Saut** : `Z`
- **Attaque Normale** : `1` (Pavé numérique ou haut)
- **Attaque Spéciale** : `2`
- **Changer de Skin** : `S` (Écran de sélection)

### Joueur 2 (P2)
- **Déplacement** : `Flèches Gauche / Droite`
- **Saut** : `Flèche Haut`
- **Attaque Normale** : `K`
- **Attaque Spéciale** : `L`
- **Changer de Skin** : `Flèche Bas` (Écran de sélection)

### Raccourcis Globaux
- **F11** : Basculer en mode Plein Écran
- **M** : Activer/Désactiver le son (Menu Principal)
- **7** : Activer/Désactiver le son (Paramètres)
- **ECHAP** : Retour au menu / Quitter la réassignation des touches
- **TAB** : Changer de carte (Écran de sélection)

## 🛠️ Installation & Lancement

1. Assurez-vous d'avoir **Python 3** d'installé.
2. Installez la bibliothèque Pygame :
   ```bash
   pip install pygame
   ```
3. Lancez le jeu :
   ```bash
   python3 main.py
   ```

## ⚙️ Paramètres
Vous pouvez réassigner toutes les touches du jeu dans le menu **[5] PARAMETRES**. Le jeu s'adapte à votre style de jeu !
